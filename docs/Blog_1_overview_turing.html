GSoC 2020: Blog 1: Overview of Turing

This series of blogs is an exploration of the turing.ml verse. This first blog briefly desribes Turing, followed by subsequent blogs which explain various functionalities and available alogrithms in Turing.

Turing.jl is a Julia package for performing Bayesian inference with probabilistic programming.

Once the package is installed in Julia (] add Turing#master), it can be used to write specific models. Here is an example:

  1. First load the package in your environment.

using Turing

  1. Define your model (say a Normal model with unknown mean and variance).

@model gdemo(x ,y) = begin
  sigma2 ~ InverseGamma(4, 3)
  mu ~ Normal(2, sqrt(sigma2))
  x ~ Normal(mu, sqrt(sigma2))
  y ~ Normal(mu, sqrt(sigma2))
end

DynamicPPL.ModelGen{Main.##WeaveSandBox#399.var"###generator#421",(:x, :y),
(),Tuple{}}(Main.##WeaveSandBox#399.##generator#421, NamedTuple())

  1. Run a sampler and collect the results. (HMC is explored more here

(a future blog in this series!)).

chn = sample(gdemo(1.2, 3), HMC(0.08, 3), 1500)
